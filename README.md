# 🌳 Inserção Recursiva em Árvore Binária (Java)

## 📌 Sobre o código

Este algoritmo implementa uma **Árvore Binária de Busca (BST)** com inserção utilizando **recursividade**.

A regra da árvore é simples:
- Valores **menores** ficam à esquerda
- Valores **maiores** ficam à direita
